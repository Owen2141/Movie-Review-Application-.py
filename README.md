# 🎬 Movie Mood Analyzer

## Overview

Movie Mood Analyzer is a Python-based Natural Language Processing (NLP) application that analyzes movie reviews and determines whether the overall sentiment is **Positive**, **Negative**, or **Neutral**.

The application retrieves movie reviews from the **TMDB API**, performs sentiment analysis using **TextBlob**, and displays the results in a user-friendly **Tkinter GUI** with a sentiment distribution chart.

This project was developed as part of a Form 6 Software Development assessment.

---

## Features

- Search for a movie by title
- Retrieve reviews using the TMDB API
- Perform sentiment analysis on reviews
- Classify reviews as Positive, Negative, or Neutral
- Display results in a graphical user interface
- Generate a sentiment pie chart
- Save results to CSV files
- Responsive interface using multithreading
